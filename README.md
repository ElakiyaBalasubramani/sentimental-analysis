# Sentiment Analysis 

## 📌 Project Overview

This project performs **Sentiment Analysis** on text data to determine whether the sentiment of a sentence is **Positive or Negative**.
It uses **Natural Language Processing (NLP)** techniques with a **Logistic Regression model** and **TF-IDF Vectorizer** for text classification.

## 🚀 Features

* Predicts sentiment of user input text
* Uses Machine Learning for classification
* Simple and lightweight Python implementation
* Demonstrates basic NLP concepts

## 🛠 Technologies Used

* Python
* Scikit-learn
* TF-IDF Vectorizer
* Logistic Regression

## ⚙️ Installation

1. Clone the repository

git clone https://github.com/yourusername/Sentiment-Analysis.git

2. Install required libraries

pip install scikit-learn pandas numpy

3. Run the application

python app.py

## 💡 How It Works

1. User enters a text sentence.
2. The text is converted into numerical features using TF-IDF.
3. The trained Logistic Regression model analyzes the features.
4. The model predicts whether the sentiment is **Positive or Negative**.
